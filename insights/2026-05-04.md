## 1. OpenAI GPT-5.5 Codex 部署于 NVIDIA GB200 NVL72 集群

【快讯内容】
OpenAI 与 NVIDIA 完成首个 10 万 GPU GB200 NVL72 集群联合调试，GPT-5.5 驱动的 Codex 实现每百万 token 成本降 35 倍、每瓦输出吞吐提升 50 倍，超万名 NVIDIA 员工已投入生产使用。

【观点】
GB200 NVL72 rack-scale 架构在 10 万 GPU 规模下验证了系统级可靠性，表明 NVLink 域内互联 + Spectrum-X 域间网络的分层拓扑已成为超大规模推理集群的事实标准。35 倍成本下降的核心来自多芯片调度与推理引擎（如 TensorRT-LLM）的内核级优化协同，对系统软件栈的固件/驱动热升级与故障隔离能力提出更高要求。

参考文章：
- [OpenAI's New GPT-5.5 Powers Codex on NVIDIA Infrastructure](https://blogs.nvidia.com/blog/openai-codex-gpt-5-5-ai-agents/)

## 2. NVIDIA 发布 Nemotron 3 Nano Omni 多模态开源模型

【快讯内容】
Nemotron 3 Nano Omni 采用 30B-A3B 混合 MoE 架构（每次前向仅激活 3B 参数），统一视觉/音频/文本推理，吞吐量为同等交互能力开源模型的 9 倍，支持 vLLM 高吞吐部署和 GGUF 端侧运行。

【观点】
30B-A3B 的稀疏激活设计使其可在 Jetson Thor/DGX Spark 等嵌入式/桌面级设备上运行完整多模态推理，是边缘 AI Agent 的重要里程碑。vLLM 的 FlashInfer 后端与 FP8/NVFP4 量化路径进一步压缩显存与延迟，对嵌入式系统软件的内存管理和实时调度提出新需求。

参考文章：
- [NVIDIA Launches Nemotron 3 Nano Omni Model](https://blogs.nvidia.com/blog/nemotron-3-nano-omni-multimodal-ai-agents/)
- [NVIDIA Nemotron 3 Nano Omni Powers Multimodal Agent Reasoning](https://developer.nvidia.com/blog/nvidia-nemotron-3-nano-omni-powers-multimodal-agent-reasoning-in-a-single-efficient-open-model/)

## 4. 利用 AI 物理加速模块化核反应堆设计

【快讯内容】
NVIDIA 开发者博客介绍使用 PhysicsNeMo 框架与 CUDA-X 库加速小型模块化反应堆（SMR）和第四代反应堆的热工水力仿真，实现实时数字孪生级别的 GPU 加速模拟。

【观点】
PhysicsNeMo 的 GPU 加速 surrogate 建模对系统软件提出高密度计算调度需求——多物理场耦合仿真需要精细的 GPU 内存分区与流水编排。该场景也是 Spectrum-X 高带宽低抖动网络的典型用例，多节点并行仿真对 NCCL/SHARP 集合通信优化路径有直接依赖。

参考文章：
- [Accelerate Clean, Modular, Nuclear Reactor Design with AI Physics](https://developer.nvidia.com/blog/accelerate-clean-modular-nuclear-reactor-design-with-ai-physics/)

## 6. DGX Spark 软件与模型优化加速本地推理

【快讯内容】
NVIDIA 发布 DGX Spark 新软件优化，llama.cpp 和 Ollama 获得 NVFP4/FP8 量化、GPU token 采样、并发改进与内存管理提升，token 生成速度提升最高 35%，同时支持 Brev 混合本地/云端路由部署。

【观点】
NVFP4 4-bit 量化首次在桌面级 GB10 芯片上实现实用化，对嵌入式推理栈（特别是量化内核与内存带宽优化）有直接参考意义。Brev 的混合路由层按数据敏感度分流本地/云端模型，体现了边缘-云协同场景下系统软件需要的安全隔离与运行时调度能力。

参考文章：
- [New Software and Model Optimizations Supercharge NVIDIA DGX Spark](https://developer.nvidia.com/blog/new-software-and-model-optimizations-supercharge-nvidia-dgx-spark/)
