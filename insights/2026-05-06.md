# NVIDIA 洞察快讯（2026年5月6日）

## 信息来源

## 快讯

【快讯内容】
NVIDIA连续发布Nemotron 3系列模型及车载AI技术栈：Nemotron 3 Super（120B/12B-active Mamba-Transformer MoE）和Nano Omni（30B-A3B多模态边缘模型）面向本地推理；DRIVE AGX Thor搭载DriveOS 7实现车载AI Agent从云到车部署；DGX Spark通过NVFP4量化与投机解码实现2.5倍性能提升。

【观点】
系统软件方面：Nemotron 3 Super的Mamba-Transformer混合架构与Latent MoE对推理引擎内核调度有重大参考价值；DGX Spark的vLLM/SGLang/TensorRT-LLM全栈优化与NVFP4量化路径直接影响推理服务端性能调优；NemoClaw/OpenShell的沙箱执行框架为Agent运行时安全提供了参考范式。嵌入式方面：DRIVE AGX Thor基于Blackwell GPU + DriveOS 7支持QNX/Linux多VM隔离，是Safety-critical嵌入式AI系统的典型架构；Nano Omni 30B-A3B的MoE稀疏激活实现9倍吞吐提升，适合Jetson等边缘部署。网络方面：DGX Spark新增ConnectX-7热插拔支持节能18W，属网络适配器管理优化，其余内容对网络方向参考价值有限。
