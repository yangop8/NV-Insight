## 1. 新 AI 软件解锁科学发现：从材料模拟到实验天文学（ISC 2026）

【快讯内容】
NVIDIA 在 ISC 2026 推出加速科学的新软件：DAQIRI 库、ALCHEMI NIM 微服务及即将发布的 cuPhoton 参考代码，把过去 CPU 上需数小时的工作变为 GPU 实时管线。Lila 材料筛选提速 50x，cuPhoton 分析天文 FITS 数据最高提速 14,900x，CERN A-GHOST 借 AI 捕捉暗物质罕见信号。

【观点】
面向系统软件：CUDA-X 进一步从 AI 训推扩展到科学计算实时流水线，DAQIRI/ALCHEMI/cuPhoton 体现"以软件吃掉专用硬件管线"的趋势——零拷贝直入 GPU、NIM 微服务化封装，降低 HPC 实验数据处理的部署门槛，对运行时调度与内存带宽利用提出更高要求。

> 参考文章：[From Materials Simulation to Experimental Astronomy, New NVIDIA AI Software Unlocks Scientific Discoveries](https://blogs.nvidia.com/blog/ai-for-science-software-cuda/)

## 2. CCCL Runtime：面向 CUDA 的现代 C++ 运行时

【快讯内容】
CCCL 3.2 引入全新惯用 C++ 接口，以 `<cuda/stream>`、`<cuda/buffer>`、`<cuda/launch>` 等头文件实现流管理、内存分配、内核启动等核心 CUDA 功能，对齐现代 C++ 设计，使 CUDA C++ 开发更安全便捷，成为传统 Runtime/Driver API 的现代化替代。

【观点】
面向系统软件：这是 GPU 系统编程基础设施的关键演进——以 RAII/类型安全的 C++ 抽象替代裸 C 风格 API，可显著减少流、内存生命周期管理中的资源泄漏与未定义行为，利好推理内核、调度器等底层组件的工程化与可维护性。

> 参考文章：[CCCL Runtime: A Modern C++ Runtime for CUDA](https://developer.nvidia.com/blog/cccl-runtime-a-modern-c-runtime-for-cuda/)

## 3. DAQIRI：为高速数据采集启用实时 AI

【快讯内容】
DAQIRI 提供软件中心化高吞吐管线，绕过传统硬件瓶颈，将高带宽探测器数据零拷贝直流入 GPU 显存做实时 AI 处理。集成 Holoscan 平台（含 TensorRT、nvCOMP），自动将网络包批处理为 GPU 张量，支持 YAML 配置与 C++/Python API。CERN A-GHOST 用其连接 FPGA 与 GPU 集群实时分析 HL-LHC 数据流。

【观点】
面向嵌入式软件与系统软件：DAQIRI 是边缘/实验现场实时推理的典型范式——零拷贝、低延迟、网络包直转张量，把网络协议处理与张量批处理下沉到框架层。Holoscan+FPGA 直连 GPU 的链路对低延迟传感数据通路、确定性调度具有参考价值。

> 参考文章：[Enable Real-Time AI for High-Speed Data Acquisition with DAQIRI](https://developer.nvidia.com/blog/enable-real-time-ai-for-high-speed-data-acquisition-with-daqiri/)

## 4. 走进 NVIDIA Halos for Robotics：面向物理 AI 的全栈功能安全系统

【快讯内容】
NVIDIA 将 Halos 自动驾驶安全体系扩展至机器人与物理 AI，构成业界首个全栈开放机器人安全系统：涵盖 IGX Thor、Holoscan Sensor Bridge 工业级 AI 计算，Halos OS（含 Halos Core 与 Outside-In 安全蓝图）。共享 ISO 26262/IEC 61508/ISO 13849 标准，经 TÜV SÜD/Rheinland 评估。Agility 的 Digit 人形机器人已采用。

【观点】
面向嵌入式软件：Halos 把安全关键系统认证（功能安全标准、第三方评估、安全评估代码行数）系统化引入机器人栈，IGX Thor + Holoscan Sensor Bridge 提供工业级实时安全计算基座。对安全关键嵌入式开发的认证路径、外部相机+AI 代理的"外向内"安全监控架构极具借鉴意义。

> 参考文章：[Inside NVIDIA Halos for Robotics: A Full-Stack Functional Safety System for Physical AI](https://developer.nvidia.com/blog/inside-nvidia-halos-for-robotics-a-full-stack-functional-safety-system-for-physical-ai/)
