## 1. Nemotron 3 Ultra：面向长程智能体的高吞吐开放模型

【快讯内容】
NVIDIA 发布 Nemotron 3 Ultra（550B MoE / 55B 激活），6月4日开放权重上线 HuggingFace、OpenRouter 与 NIM。采用混合 Mamba-Transformer 与 NVFP4 量化，吞吐较同级开放模型最高提升约5倍，长程智能体任务成本降低约30%。

【观点】
- **系统软件**：约85%层使用 Blackwell 原生 NVFP4、注意力/投影层保留 BF16/MXFP8 的混合精度，是推理内核与量化协同设计的范例；多 token 预测与 LatentMoE 路由直接降低多轮 agent 的 token 开销，对 vLLM/SGLang/TensorRT/Dynamo 部署调度有现实意义。
- **嵌入式软件**：4-bit 权重大幅压缩显存占用，为后续向边缘/Jetson 蒸馏出小尺寸专用模型提供基座；其多教师在线蒸馏配方利于端侧领域定制。
- **网络**：550B 级 MoE 的专家并行依赖高带宽 scale-up 互联（NVLink/Spectrum-X），印证 token 工厂网络拓扑对吞吐的决定性影响。

> 参考文章：[NVIDIA Nemotron 3 Ultra Powers Faster, More Efficient Reasoning for Long-Running Agents](https://developer.nvidia.com/blog/nvidia-nemotron-3-ultra-powers-faster-more-efficient-reasoning-for-long-running-agents/)

## 2. Windows 端侧智能体：MXC 沙箱 + NVIDIA OpenShell 运行时

【快讯内容】
NVIDIA 联合微软在 Windows 推出端侧 AI 智能体安全栈：微软 MXC（eXecution Containers）基于原生 OS 构件提供隔离/包含策略层，NVIDIA OpenShell 运行时构建其上，实现策略管理、推理路由与 PII 脱敏，并带来约2倍 agentic 推理加速。

【观点】
- **系统软件**：每个 agent 运行于独立沙箱、出站调用在到达文件/网络/凭据前按策略评估，是运行时安全（进程隔离、最小权限、prompt injection 防护）落到操作系统原生构件的工程样板；OpenShell 的二进制/目标/方法/路径四级评估值得借鉴。
- **嵌入式软件**：端侧 always-on agent 推理路由可按隐私策略就近选择本地小模型，契合端云协同与内存优化的部署范式。
- **网络**：PII 脱敏后再发往云端模型，本质是数据出域前的零信任策略执行，与企业 AI 工厂的零信任架构思路一致。

> 参考文章：[Build Personal AI Agents on Windows PCs with New Tools from Microsoft and NVIDIA](https://developer.nvidia.com/blog/build-personal-ai-agents-on-windows-pcs-with-new-tools-from-microsoft-and-nvidia/)
