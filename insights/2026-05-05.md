## 1. NVIDIA Nemotron 3 Nano Omni：统一视觉/音频/语言的全模态开源模型

【快讯内容】
NVIDIA 发布 Nemotron 3 Nano Omni，基于 30B-A3B 混合 MoE 架构，单模型统一视频、音频、图像和文本推理，吞吐量较同类开放全模态模型高 9 倍，支持从 Jetson 到数据中心的全栈部署。

【观点】
该模型对嵌入式软件方向有重要参考价值：30B 总参数仅 3B 激活的 MoE 设计大幅降低边缘推理算力需求，明确支持 Jetson 和 DGX Spark 本地部署，使端侧多模态智能体成为可行方案。对系统软件方向，其 MoE 专家动态激活机制对推理引擎内核调度（专家路由、内存管理）提出新优化需求，vLLM/SGLang 等引擎需适配此类架构的高效 serving。

- [NVIDIA Nemotron 3 Nano Omni Powers Multimodal Agent Reasoning in a Single Efficient Open Model](https://developer.nvidia.com/blog/nvidia-nemotron-3-nano-omni-powers-multimodal-agent-reasoning-in-a-single-efficient-open-model/)

## 2. NVIDIA Dynamo 1.0：面向 AI 工厂的分布式推理操作系统正式投产

【快讯内容】
NVIDIA Dynamo 1.0 正式发布，作为 AI 工厂的开源推理操作系统，在 GB200 NVL72 上通过解耦式 serving 实现 7 倍性能提升，已被 CoreWeave、ByteDance 等用于生产环境，集成 vLLM、SGLang、TensorRT-LLM 等主流引擎。

【观点】
对系统软件方向有高参考价值：Dynamo 1.0 的核心创新在于多节点 GPU 资源编排——智能路由请求至已缓存 KV Cache 的 GPU、跨节点内存卸载、解耦 prefill/decode 阶段。其 NIXL 组件被 vLLM/SGLang 集成用于加速 KV Cache 跨 GPU 传输，代表了推理引擎内核优化从单节点走向集群级调度的范式转变。对网络方向，Dynamo 的跨节点通信依赖高带宽低延迟互联，是 Spectrum-X 以太网平台的典型应用场景。

**参考文章：**
- [How NVIDIA Dynamo 1.0 Powers Multi-Node Inference at Production Scale](https://developer.nvidia.com/blog/nvidia-dynamo-1-production-ready/)
- [NVIDIA Enters Production With Dynamo, the Broadly Adopted Inference Operating System for AI Factories](https://nvidianews.nvidia.com/news/dynamo-1-0)

## 3. NVIDIA Nemotron 3 Super：Mamba-Transformer 混合 MoE 实现 5 倍推理吞吐

【快讯内容】
NVIDIA 发布 Nemotron 3 Super，120B 参数/12B 激活的混合 Mamba-Transformer MoE 开源模型，Mamba 层提供 4 倍内存和计算效率，Latent MoE 以单专家代价激活四个专家，面向复杂多智能体系统中的高频执行任务。

【观点】
对系统软件方向有参考价值：Mamba 状态空间模型层与 Transformer 注意力层的混合架构对推理引擎提出新挑战——需要同时支持线性复杂度的 Mamba 状态递推和二次复杂度的注意力计算，TensorRT-LLM 等引擎的内核调度需针对此类混合架构做算子融合和内存布局优化。120B 参数仅 12B 激活的极端稀疏性对 MoE 路由和多节点分片策略也有新要求。

**参考文章：**
- [New NVIDIA Nemotron 3 Super Delivers 5x Higher Throughput for Agentic AI](https://blogs.nvidia.com/blog/nemotron-3-super-agentic-ai/)
- [Introducing Nemotron 3 Super: An Open Hybrid Mamba-Transformer MoE for Agentic Reasoning](https://developer.nvidia.com/blog/introducing-nemotron-3-super-an-open-hybrid-mamba-transformer-moe-for-agentic-reasoning/)
