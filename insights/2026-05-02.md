## 1. OpenAI GPT-5.5 基于 NVIDIA 基础设施运行 Codex

【快讯内容】
OpenAI 发布 GPT-5.5 并驱动 Codex 编程代理，部署于 NVIDIA GB200/GB300 NVL72 机架级系统，实现每百万 token 成本降低 35 倍、每兆瓦 token 产出提升 50 倍。OpenAI 已承诺部署超过 10GW 的 NVIDIA 系统用于下一代 AI 基础设施。

【观点】
GB200 NVL72 机架级部署对网络提出极高要求——72 GPU 节点内部依赖 NVLink 全互连，跨机架通信依赖 Spectrum-X/InfiniBand 高带宽低延迟交换。10GW 规模的基础设施意味着百万 GPU 级别的集群调度和网络拓扑管理成为关键瓶颈，对多芯片/多节点调度、推理引擎内核优化（50x 吞吐提升暗示 vLLM/TensorRT 级别的 kernel 重构）及系统级功耗管理均有直接参考价值。

参考文章：
- [OpenAI's New GPT-5.5 Powers Codex on NVIDIA Infrastructure](https://blogs.nvidia.com/blog/openai-codex-gpt-5-5-ai-agents/)

## 2. NVIDIA Nemotron 3 Nano Omni：统一视觉/音频/语言的边缘多模态模型

【快讯内容】
NVIDIA 发布 Nemotron 3 Nano Omni 开源多模态模型，采用 30B-A3B 混合专家架构（每次前向仅激活 3B 参数），支持视觉、音频、图像、文本统一推理，吞吐量达同类模型 9 倍。可部署于 Jetson、DGX Spark 到数据中心全场景。

【观点】
该模型对嵌入式软件方向意义重大：30B 总参数、3B 活跃参数的 MoE 设计专为 Jetson 等边缘平台优化，是端侧多模态代理落地的关键进展。模型量化与内存优化技术（单 GPU 运行 30B 模型）、实时多模态推理调度（音视频+文本同步处理）对实时操作系统和安全关键系统场景（工业机器人、医疗设备）有直接工程参考。vLLM 已提供对该模型的原生支持，推理引擎内核优化同步推进。

参考文章：
- [NVIDIA Launches Nemotron 3 Nano Omni Model](https://blogs.nvidia.com/blog/nemotron-3-nano-omni-multimodal-ai-agents/)
- [NVIDIA Nemotron 3 Nano Omni Powers Multimodal Agent Reasoning](https://developer.nvidia.com/blog/nvidia-nemotron-3-nano-omni-powers-multimodal-agent-reasoning-in-a-single-efficient-open-model/)

## 3. NVIDIA Ising：面向容错量子系统的 AI 工作流

【观点】
