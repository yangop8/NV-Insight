## 一、Vera CPU 满载吞吐再拆解：为智能体 AI 工厂的 RL 训练与推理提速

【快讯内容】
Vera CPU 采单片计算 die、88 Olympus 核、统一缓存与二代 Scalable Coherency Fabric，满 socket 负载下持续单核性能达传统 x86 的 1.8x、峰值加载时延低 40%、单核内存带宽 3x 且功耗不到一半。直接提升 RL 训练吞吐、策略梯度质量与环境 rollout 完成率；靠减少 CPU 侧停顿与 KV-cache 逐出后的上下文重建，抬高 GPU 利用率与 SLA 达成。

【观点】
面向系统软件：命中智能体/RL 的真实瓶颈——分支密集的串行 CPU 关键路径。单核吞吐与低时延缓存直接决定 rollout 与策略评估的完成率，"减少 KV-cache 逐出后的上下文重建"更是对推理引擎显存-上下文调度的正面呼应，是 GPU 利用率优化的 CPU 侧抓手。面向网络：满载低时延与高单核内存带宽利于 CPU-GPU-NVLink/IB 协同关键路径，降低密集智能体工厂的尾时延。面向嵌入式：本文面向数据中心，端侧参考价值间接。
