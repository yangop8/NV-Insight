## 1. NVIDIA 全栈推理软件栈如何撑起最低 token 成本

【快讯内容】
NVIDIA 详解全栈推理软件栈三层架构（生产运营/应用加速/基础设施访问）：凭 NVFP4、多 token 预测、大专家并行与分离式服务，将 Blackwell 单 GPU 吞吐提升至 20x，一个月内使 DeepSeek V4 单 token 成本降至 1/5，凸显软件迭代红利可比肩硬件换代。

【观点】
系统软件方向：推理引擎内核优化（vLLM/SGLang/TensorRT/Dynamo）与运行时编排是"软件即性能"的核心，"一个月 5x 降本"证明软件栈迭代的复利可与硬件代际相当，是 token 经济的主战场。网络方向：分离式服务（prefill/decode 分离）与大专家并行强依赖 NVLink/InfiniBand 低延迟高带宽互联与 KV 感知路由，网络正成为吞吐与成本的隐性瓶颈。嵌入式方向参考有限。

> 参考文章：[How NVIDIA's Inference Software Stack Powers the Lowest Token Cost](https://blogs.nvidia.com/blog/inference-software-lowest-token-cost/)

## 2. 硬件信任根的 AI 安全：几乎零性能损失

【快讯内容】
NVIDIA 机密计算实测：Blackwell 硬件信任根 + NVLink 加密 + NRAS 远程证明，在 HGX B300 上跑 Qwen3.5-397B FP8，开启 CC 后吞吐/延迟开销普遍 <8%；借 FlashInfer 安全自调优、异步 D2H 拷贝、SGLang 分段 CUDA Graph，加密推理逼近原生性能。

【观点】
网络方向：NVLink 链路加密 + 远程证明（NRAS）把零信任从主机边界扩展到 GPU 互联层，是机密 AI 工厂内"数据在途"保护与多租户隔离的关键拼图。系统软件方向：CC 安全自调优 / 异步拷贝 / 分段 CUDA Graph 把运行时安全（可信执行、远程证明）的开销压到 <8%，可用性拐点已至，为金融、医疗、政务等受监管推理落地扫清性能顾虑。嵌入式方向：端侧安全关键推理可借鉴"硬件信任根 + 低开销证明"的设计范式。

> 参考文章：[Hardware-Rooted AI Security That Won't Slow You Down](https://developer.nvidia.com/blog/hardware-rooted-ai-security-that-wont-slow-you-down)

## 3.（附录）比热水浴还热：45°C 液冷冷却 AI 巨型机器

【快讯内容】
